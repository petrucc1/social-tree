# Social tree

Este projeto foi realizado para cumprir o desafio de criar uma página com uma lista de links (menu) que pode ser usada em perfis de redes sociais para direcionar o usuário para o local escolhido. Aproveitei do desafio ser nível iniciante e me desafiei aprendendo e aplicando a folha de estilo SCSS, consegui concluir com sucesso utilizando as variáveis, o aninhamento e o nth-last-child do próprio CSS3.

## Tecnologias usadas
* HTML5
* SCSS
